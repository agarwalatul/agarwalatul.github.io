# Atul Agarwal - Professional Profile

This repository contains the source code for the professional profile and portfolio of **Atul Agarwal**, a Quantitative Technologist specializing in Data Engineering and Machine Learning.

## Project Overview

The project is a high-performance, responsive static web application designed to showcase professional experience, technical skills, and featured projects. It is built as a lightweight alternative to heavy profile builders, focusing on speed, clean aesthetics, and automated deployment.

### Key Highlights
- **Role**: Quantitative Technologist | Data Engineering & ML
- **Experience**: 7.5+ years in trading data systems and scalable ML platforms.
- **Tech Stack**: Python, Rust, Kafka, ClickHouse, AWS, Docker, Kubernetes.
- **Goal**: To provide a professional, print-friendly digital resume with seamless updates via a data-driven configuration.
